##短網址產生器
將網址做個轉換

###功能
*將輸入的網址轉換成短網址
*透過短網址連結到特定(原本輸入)的頁面

###開始使用
若無 node.js 與 npm 請先安裝
↓
將專案 clone 到本地
↓
在本地開啟後，透過終端機進入資料夾，輸入：
```bash
npm install
```
↓
安裝完畢後，繼續輸入：
```bash
npm run start
```
↓
看到 
```bash
App is running on http://localhost:3000
mongodb connected!
```
表示伺服器完成啟動, 進入 http://localhost:3000 開始使用

###結束使用
```bash
ctrl + c
```

##開發工具

*Express 4.17.1

*MongoDB
